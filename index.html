issnow Launcher の使い方ガイド

起動方法

① トップページから好きなバージョンを選択
② 「Play」をクリック
③ ロードが終わるまで待つ

LANの中継方法

① esc(エスケープキー)を押してPauseメニューを開いて下さい。
② Invite(招待)をクリック
③ 名前は何でも大丈夫。(LANに公開したときの名前になります。)もし、友達とかだけでやりたいな〜って時は必ずHidden(隠す)を押して下さい。(チーターなどの参加があるかもしれません。AllowCheatは十分にご検討後ON,OFFの切り替えをお願いします。)
④5桁のコードを友だちに教えるか、マルチプレイのところにLAN Networkなどで表示された物をクリックするだけで参加できます。

🌐 サーバー参加方法

① Multiplayer を選択
② Server Address を入力
③ Join Server をクリック

⚠ よくある問題

  • ロードが止まる → ページ再読み込み
  • 重い → WASM-GCを使う
  • 起動しない → Chrome推奨(flagのJSPIを使うのがおすすめ。)

Hint

  • 軽さ重視 → 1.8.8 WASM-GC
  • 機能重視 → 1.12.2
  • PvP → 1.8.8,PixelClient,ResentClient等がおすすめです。
